# Code Book

This code book summarizes `tidy_data.txt`, which is created using the script `run_analysis.R` and data collected from the accelerometers from the Samsung Galaxy S smartphone, which can be found [here](https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip). All measurement variables in `tidy_data.txt` are means of mean and std measurements from the original data set by below stated identification variables.

## Identification variables

- `Subject_ID` - identifies the subject from which the measurement was taken
- `Activity` - identifies the activity performed when measurement was taken:
  - 1. WALKING
  - 2. WALKING_UPSTAIRS
  - 3. WALKING_DOWNSTAIRS
  - 4. SITTING
  - 5. STANDING
  - 6. LAYING
